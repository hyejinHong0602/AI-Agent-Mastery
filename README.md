
# 🤖 AI Agent Engineering Study

> **"지능형 에이전트 설계부터 구현까지"** > 『AI 에이전트 엔지니어링』 도서를 학습하며, 매일 코드로 증명하고 기록하는 레포지토리

## 📅 학습 목표
- **1일 1커밋**: 매일 에이전트 설계 패턴 구현 또는 이론 정리
- **실전 적용**: 책의 예제를 넘어 나만의 도구(Tool)와 워크플로우 결합
- **최적화 고민**: 프롬프트 엔지니어링 및 토큰 효율성 고려

---

## 🛠 Tech Stack
- **Language**: Python
- **Framework**: LangChain / LangGraph
- **Model**: OpenAI GPT-4o / Claude 3.5 Sonnet
- **Tools**: Tavily Search, Python Interpreter, etc.

## 🪵 커밋 컨벤션 (Commit Message)
효율적인 이력 관리를 위해 아래 규칙을 준수합니다.
- `feat`: 새로운 에이전트 기능 또는 도구 구현
- `study`: 이론 정리 및 Markdown 문서 업데이트
- `fix`: 코드 오류 수정 및 프롬프트 튜닝
- `refactor`: 코드 구조 개선 및 최적화
